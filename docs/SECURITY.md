# 보안

## 환경 변수 관리

### 필수 환경 변수

| 변수 | 용도 | 저장 위치 |
|------|------|----------|
| `VITE_GA_MEASUREMENT_ID` | Google Analytics 4 측정 ID | `.env` |

### 규칙

1. `.env` 파일은 **절대 커밋하지 않는다** (`.gitignore`에 포함)
2. `.env.example`에는 키 이름만 기록하고, 실제 값은 비워둔다
3. CI/CD 환경에서는 시크릿 매니저 또는 환경 변수 주입으로 관리
4. 로컬 개발 시 `.env` 파일에서 로드

## 데이터 보호

### 외부 데이터 파일
- 다운로드/생성된 데이터 파일은 `data/` 디렉토리에 저장
- `data/` 디렉토리는 `.gitignore`에 포함

### 인젝션 방지
- DB 쿼리: 파라미터 바인딩 또는 ORM/쿼리빌더만 사용. 문자열 연결로 쿼리 구성 금지
- 사용자 입력: 경계에서 검증 후 정규화하여 사용

### API 키 노출 방지
- 에러 메시지에 API 키, URL 등 민감 정보를 포함하지 않는다
- 로그에 시크릿을 출력하지 않는다
- 클라이언트 응답에 내부 에러 스택 트레이스를 포함하지 않는다
