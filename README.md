# Blue Badge Remover Landing

[blue-badge.fotone.moe](https://blue-badge.fotone.moe)

X(Twitter)에서 유료 파란 뱃지 계정의 트윗을 자동으로 숨기는 Chrome 확장 프로그램 **Blue Badge Remover**의 랜딩 페이지.

## 관련 링크

- [Chrome Web Store](https://chromewebstore.google.com/detail/Blue%20Badge%20Remover/cjhmbgfnddpcdfmoicfcocekmainhhdm)
- [확장 프로그램 소스코드](https://github.com/fotoner/blue-badge-remover)

## 기술 스택

- React 19 + TypeScript
- Tailwind CSS 4
- TanStack Router (파일 기반)
- Vite
- Vitest + React Testing Library
- GitHub Pages

## 개발

```bash
npm install
npm run dev      # http://localhost:5173
npm test         # vitest
npm run build    # 프로덕션 빌드
```
